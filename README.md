# LoveLive! School idol STORE 定时购物车

一个用于 [LoveLive! 官方商店](https://lovelive.fannect.jp/) 的 Tampermonkey / Violentmonkey 用户脚本，可在任意集合（Collections）页面预选商品及角色变体，并设定时间通过店铺原生接口自动加入购物车。

> **适用于抢购 / 限时开售场景。** 可提前勾选尚未开售的变体，到达设定时间后自动提交加车请求并跳转至购物车页面。

---

## ✨ 功能特性

- **读取当前集合页所有商品变体** — 自动解析页面中的商品链接，通过 Shopify `.js` 接口获取完整变体信息（价格、库存状态、图片等）。
- **变体级勾选 & 数量设置** — 每个角色 / 规格变体独立显示，可分别勾选并设定购买数量。
- **定时自动加车** — 设定目标时间后，脚本会在到达时间时发起一次 `POST /cart/add.js` 请求，成功后自动跳转至购物车。
- **待开售变体可预选** — 即使变体当前显示为"待开放/无库存"，也可提前勾选，开售后由定时器自动提交。
- **拖拽浮动按钮** — 右下角的「定时购物车」按钮可自由拖拽定位，位置自动记忆。
- **毛玻璃 UI 面板** — 半透明毛玻璃风格面板，支持滚动，适配移动端响应式布局。
- **全选 / 取消全选** — 一键切换全部变体的勾选状态。
- **本地持久化** — 选中的变体、数量、定时时间均保存在 `localStorage`，刷新页面不丢失。

## 📦 安装

1. 确保浏览器已安装用户脚本管理器扩展：
   - [Tampermonkey](https://www.tampermonkey.net/)（推荐）
   - [Violentmonkey](https://violentmonkey.github.io/)
2. 点击脚本文件 [`LoveLive School idol STORE 购物车.js`](./LoveLive%20School%20idol%20STORE%20购物车.js) 查看源码，复制全部内容。
3. 在脚本管理器中「新建脚本」，粘贴代码并保存。
4. 访问任意 LoveLive! 官方商店集合页（如 `https://lovelive.fannect.jp/collections/...`），页面右下角会出现「定时购物车」按钮。

## 🚀 使用说明

| 步骤 | 操作 |
|------|------|
| 1 | 打开任意集合页面，点击右下角 **「定时购物车」** 按钮打开面板。 |
| 2 | 面板自动读取当前页面所有商品变体；如未自动加载，点击 **「刷新商品」**。 |
| 3 | 勾选需要购买的商品变体，设定数量（默认 1）。 |
| 4 | 在 **「加车时间」** 输入框中设定目标时间（使用浏览器本地时区）。 |
| 5 | 点击 **「保存并设定定时器」**。状态栏会显示已暂存的变体数量及计划执行时间。 |
| 6 | **保持标签页打开**，到达设定时间后脚本会自动提交加车请求并跳转至购物车。 |

> ⚠️ **重要提示：** 定时器依赖浏览器标签页保持打开且处于前台（或至少不被系统休眠）。建议在设定时间前确保设备不休眠、浏览器不关闭该标签页。

## 🔧 技术细节

| 项目 | 说明 |
|------|------|
| **匹配站点** | `https://lovelive.fannect.jp/collections/*` |
| **数据接口** | Shopify 原生 `/products/{handle}.js` 与 `/cart/add.js` |
| **数据存储** | `localStorage`（键名 `lovelive_cart_schedule_v1` 和 `lovelive_cart_button_position_v1`） |
| **最大定时延迟** | 约 24.8 天（`setTimeout` 最大延迟 `2,147,483,647` ms），超时会自动分段续期 |
| **认证方式** | `credentials: 'same-origin'`，依赖浏览器已登录的会话 Cookie |

## 🧹 清除暂存

如需取消已设定的定时任务或清空已选变体：

- 打开面板，点击 **「清除暂存」** 按钮，所有勾选、数量及定时时间将被清空。
